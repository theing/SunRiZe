Editor Commands


Insert a object
 Select Module or State or Branch from the toolbar and click over the panel
Insert a junction
Select a Dependency or Transition from the toolbar and press on the starting object then on the destination object.
Select an object Press left button over the object, the selection is released when the mouse button is released.
Move a object Select and drag the object around then release the button.
Break a junction
 The junction position depends from the one/two standalone objects and cannot be moved.
To create a breaking point click over the junction and drag the mouse.
The breaking points do not depends by the objects, to move them click over and drag the mouse.
Remove a breaking point
 Click over the breaking point and press the [Back] key.
Remove an object
 Select the object by clicking on it, then press the [Del] key and confirm.
Note : if you remove a standalone object all the connected junctions are removed too.
Object Editing
 Double click over the object. If the object is editable the related Editing Dialog will appear.
Create a new  FSM in a Module
 Select a Module from the main context and press the [Ins] key.
The FSM will be created and the context switched.
Copy an object
 Select a Module or State and press [Ctrl] + C , then release the mouse and click on the panel to obtain a copy.
Undo
 Press [Ctrl] + Z to undo the last modification.
Redo
 Press [Ctrl] + X to revert the latest undo operations.
