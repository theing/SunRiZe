
SunRiZe is a simplified Model Driven Development software, used to generate C code 
for micro platforms, like Arduino.

SunRiZe is written in pure C++-1x with wxWidget and embedded javascript (http://duktape.org).

SunRiZe has been developed on Linux, but it is also available on Windows and portable to other platforms.

Go to :

https://theing.blogspot.it/p/sunrize.html 

for further information, contacts and the complete documentation.

Good luck, Marco
